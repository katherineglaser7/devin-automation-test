# TaskMaster

A simple command-line task management tool built with Python.

## Overview

TaskMaster is a CLI application for managing tasks with support for priorities, statuses, and tags.

## Quick Start

```bash
pip install -e .
taskmaster add "My first task" --priority high
taskmaster list
```

## Features

- Create, list, complete, and delete tasks
- Priority levels (low, medium, high, critical)
- Status tracking (pending, in_progress, completed, cancelled)
- Tag support for organization
- Persistent storage
